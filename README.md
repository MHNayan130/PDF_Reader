# AnyStudy

A simple static study portal demo with login, signup, password reset, and file upload preview.

## Run the project

1. Open a terminal in this folder.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the server:
   ```bash
   npm start
   ```
4. Open `http://localhost:3000` in your browser.

## Features

- Wave-style landing page
- Upload and preview files on the client
- Login, signup, and reset password pages
- Google sign-in button placeholder
- Client-side auth simulation via `localStorage`
